Konto löschen

Sie können Ihr KatchIt!-Konto jederzeit direkt in der App löschen. Bei der Löschung werden alle Ihre Daten unwiderruflich von unseren Servern entfernt.

So löschen Sie Ihr Konto

1 Einstellungen öffnen

Tippen Sie auf das Zahnrad-Symbol oben rechts auf dem Startbildschirm.

2 Konto-Bereich aufrufen

Scrollen Sie zum Bereich "Konto" in den Einstellungen.

3 "Konto löschen" wählen

Tippen Sie auf "Konto löschen" und bestätigen Sie die Löschung im Dialogfenster.

Welche Daten werden gelöscht?

DatentypAktion
E-Mail-AdresseVollständig gelöscht
AnzeigenameVollständig gelöscht
Profilbild / AvatarVollständig gelöscht
Spielstatistiken (Sterne, Erfolge)Vollständig gelöscht
FreundeslisteVollständig gelöscht
Kaufhistorie (In-App-Käufe)Vollständig gelöscht
AuthentifizierungsdatenVollständig gelöscht

Die Löschung erfolgt sofort und unwiderruflich. Nach der Löschung werden Sie automatisch abgemeldet. Ein gelöschtes Konto kann nicht wiederhergestellt werden.

Aufbewahrungsdauer

Es gibt keine zusätzliche Aufbewahrungsdauer. Alle oben genannten Daten werden sofort und vollständig von unseren Servern (Supabase, EU-Frankfurt) entfernt, sobald Sie die Löschung bestätigen.

Konto löschen ohne App-Zugang

Falls Sie keinen Zugriff mehr auf die App haben, können Sie die Löschung Ihres Kontos auch per E-Mail anfordern:

Schreiben Sie uns an

support@katchit.app

Bitte geben Sie die E-Mail-Adresse an, mit der Sie sich registriert haben. Die Löschung erfolgt innerhalb von 7 Werktagen.

Weitere Informationen zum Umgang mit Ihren Daten finden Sie in unserer Datenschutzerklärung.
